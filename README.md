
# 👨🏻‍💻Customer Behavior Data Analyst Portfolio Project

## 📊 DATA ANALYSIS PROJECT 

### 📌 Overview

The goal of this project is to simulate a corporate-grade end-to-end data analytics workflow, demonstrating the ability to translate raw data into strategic business intelligence by:

✅ Data Preparation,Modeling & Exploratory Data Analysis (Python): Clean and transform the raw dataset for analysis.

✅ Data Analysis (SQL): Simulate business transactions, and run queries to extract insights on customer segments, loyalty, and purchase drivers.

✅ Visualization & Insights (Power BI): Build an interactive dashboard that highlights key patterns and trends, enabling stakeholders to make data-driven decisions.

✅ Report and Presentation: Write a clear project report summarizing your key findings and business recommendations. Prepare a presentation that visually communicates insights and actionable recommendations to stakeholders.

### 📂 Dataset

The dataset contains customer-related information such as purchases, categories, and transaction details.
It was used to analyze patterns in customer behavior, spending, and product trends.
Data was initially loaded and processed using Python.

### 🛠️ Tools & Technologies Used

Python (Pandas, NumPy, Matplotlib, Seaborn)

SQL (PostgreSQL / MySQL / SQL Server)

Power BI Desktop (Dashboard creation)

Gamma (Presentation creation)

Jupyter Notebook / VS Code

## 🔄 Project Workflow

### 1. Data Loading

Imported dataset into Python using Pandas

Performed initial inspection (shape, columns, data types)

### 2. Data Cleaning

Handled missing values

Removed duplicates

Fixed data types

Standardized column names

### 3. Exploratory Data Analysis (EDA)

Analyzed key metrics like average purchase, total revenue

Identified trends and patterns using visualizations

Checked distributions and relationships between variables

### 4. SQL Analysis

Loaded cleaned data into database

Wrote SQL queries to:

Aggregate data (SUM, AVG, COUNT)

Group by categories

Rank top-performing items

Filter insights using conditions

### 5. Dashboard Creation

Built an interactive dashboard in Power BI

Added KPIs, charts, and filters

Focused on user-friendly and clean design

### 6. Reporting & Presentation

Created a detailed report summarizing findings

Designed a presentation using Gamma

Highlighted key insights and business recommendations

### 📊 Dashboard Highlights

KPI Cards (Total Revenue, Average Spend, Customer Count)

Category-wise and item-wise analysis

Trend visualizations

Interactive filters for better exploration
### 📈 Key Results & Insights

Identified top-performing product categories

Analyzed customer spending behavior

Found trends in purchase patterns

Highlighted areas for potential business improvement

## ▶️ How to Run the Project

### 1. Python (EDA & Cleaning)
